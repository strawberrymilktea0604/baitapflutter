# Hướng dẫn thêm hình ảnh

Để ứng dụng hiển thị đúng, bạn cần thêm 4 file hình ảnh vào thư mục này:

1. **place1.jpg** - Hình ảnh cổng Torii (Nhật Bản) - màu hồng/đỏ
2. **place2.jpg** - Hình ảnh tòa nhà/nhà thờ (màu xanh dương)
3. **place3.jpg** - Hình ảnh lâu đài (màu xanh dương/trắng)
4. **place4.jpg** - Hình ảnh đấu trường La Mã (màu vàng/cam)

## Cách thêm hình ảnh:

1. Tải hoặc tìm 4 hình ảnh địa điểm du lịch
2. Đổi tên chúng thành: place1.jpg, place2.jpg, place3.jpg, place4.jpg
3. Copy vào thư mục: `bai_1/assets/images/`

## Gợi ý tìm hình:

- Tìm kiếm trên Google Images với từ khóa: "travel destinations", "famous landmarks"
- Hoặc sử dụng các trang miễn phí như: unsplash.com, pexels.com
- Tìm hình theo các địa điểm: Japan torii gate, European cathedral, Castle, Colosseum

Sau khi thêm hình, chạy lại ứng dụng với lệnh: `flutter run`
